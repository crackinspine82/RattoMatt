# RapidFire Rules (MVP1)

RapidFire is an online, timed testing mode available in Student Mode.

## 1) Session Types
- Quick Test: 10 minutes, 20 questions
- Brief Test: 15 minutes, 30 questions
- Rally Test: 30 minutes, 60 questions

Question count rule: questions = minutes * 2.

## 2) Question Types
- Objective questions only
- No typing required for MCQ/T-F

## 3) Difficulty Mix
- Target 25% per difficulty level (easy/medium/difficult/complex)
- Best-effort if a difficulty pool is insufficient

## 4) Timing Behavior
- Session has a global timer
- Students can move back and forth before submission
- When time expires, the test auto-submits in its current state

## 5) Repeat Avoidance
- RapidFire correct answers are excluded from new tests for 60 days
- If the pool is exhausted, repeats are allowed
