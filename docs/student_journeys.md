# RattoMatt - Student User Journeys

## Journey 1: Entering Student Mode
**Goal:** Parent hands the phone to the student in distraction-free mode.
1. Parent taps "Student Mode".
2. App locks navigation and notifications.
3. Student sees two options: RapidFire and Flashcards.

## Journey 2: RapidFire Session
**Goal:** Complete a timed objective quiz session.
1. Student selects session type:
   - Quick Test (10 mins / 20 questions)
   - Brief Test (15 mins / 30 questions)
   - Rally Test (30 mins / 60 questions)
2. Student answers questions and can navigate back and forth.
3. Student taps Submit or the timer expires (auto-submit).
4. Parent re-enters with biometric/PIN to exit Student Mode.

## Journey 3: Flashcards Session
**Goal:** Quick revision in a timed, distraction-free format.
1. Student selects Flashcards.
2. Cards are shown one at a time with simple navigation.
3. Session ends when student exits or timer expires.
4. Parent re-enters with biometric/PIN to exit Student Mode.

## Parent Handoff and Handoff Back
- Parent initiates Student Mode.
- Parent exit requires biometric/PIN.
- Any session auto-submits on timeout.
